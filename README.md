# 🏦 Bank Management System (Python + Streamlit)

This project is a **Simple Bank Management System** developed using **Python**.
It provides basic banking operations such as account creation, deposit, withdrawal, and account management.

The project includes **two versions of the application**:

1. **Console-based application** written in pure Python.
2. **Web-based application** built using Streamlit.

## Features

* Create a new bank account
* Deposit money
* Withdraw money
* View account details
* Update user information
* Delete an account

## Technologies Used

* Python
* Streamlit
* JSON (for storing account data)

## Project Files

* **hello.py** – Console-based bank management system (Python only)
* **app.py** – Streamlit web interface
* **main.py** – Core banking logic and account management functions
* **data.json** – Stores account data and balances

## How to Run

### Run Console Version

```bash
python hello.py
```

### Run Streamlit Web App

```bash
pip install streamlit
streamlit run app.py or python -m streamlit run app.py
```

## Description
This project demonstrates how a basic banking system can be implemented using Python.
The Streamlit version converts the Python program into a simple interactive web application.
